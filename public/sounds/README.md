# Sound Effects

This directory contains sound effects for the Anime Randomizer application.

## Required Sound Files

Add the following MP3 files to this directory:

### spin.mp3
- Used when the random anime selector is spinning
- Recommended: Short (0.5-1s) whoosh or spin sound effect
- Format: MP3
- Size: < 100KB recommended

### success.mp3
- Played when a random anime is successfully selected
- Recommended: Pleasant notification or success chime
- Format: MP3
- Size: < 100KB recommended

## Finding Free Sound Effects

You can find royalty-free sound effects from:
- [Freesound.org](https://freesound.org)
- [Zapsplat.com](https://www.zapsplat.com)
- [Mixkit.co](https://mixkit.co/free-sound-effects/)
- [Pixabay Sounds](https://pixabay.com/sound-effects/)

## Usage

These sounds are referenced in:
- `src/components/random/RandomAnimeButton.tsx`

Make sure the file names match exactly as specified above.
